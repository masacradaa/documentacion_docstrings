Clase de configuración para la aplicación Flask.

Las variables se cargan desde el entorno usando dotenv. Si no existe la variable SECRET_KEY, se utiliza un valor por defecto.

src.crear_app()[fuente]

Crea e inicializa la aplicación Flask.

  • Carga la configuración desde Config.

  • Registra los blueprints de los distintos controladores.

  • Define un middleware after_request para evitar el cache en las respuestas.

  • Registra la ruta raíz (/) que renderiza home.html.

Devuelve:

Instancia configurada de Flask.

Tipo del valor devuelto:

flask.Flask
