src.controllers.auth_controller module

src.controllers.auth_controller.login()[fuente]

Vista para autenticar un usuario.

  • En GET: muestra el formulario de inicio de sesión.

  • En POST: valida credenciales contra el servicio de login y crea sesión si son correctas.

Devuelve:

Renderizado de la plantilla de login o redirección a la vista principal si el login es exitoso.

Tipo del valor devuelto:

str

src.controllers.expense_controller module

src.controllers.expense_controller.editar_gasto(gasto_id)[fuente]

Muestra y actualiza un gasto específico.

GET:

  • Renderiza el formulario de edición de un gasto.

POST:

  • Procesa los cambios enviados y actualiza el gasto en la base de datos.

Parámetros:

gasto_id (int) – ID del gasto a editar.

Query user_id:

ID del usuario propietario. Default = 1.

Form producto:

Nuevo nombre del producto (opcional).

Form monto:

Nuevo monto (opcional).

Form categoria:

Nueva categoría (opcional).

Form es_hormiga:

Si está presente, se marca como gasto hormiga.

Devuelve:

Render de la plantilla gastos_editar.html con: - user_id (int): ID del usuario. - gasto (Expense): Gasto a editar. - categorias (list): Categorías válidas. - errores (dict): Diccionario con errores si los hay.

Tipo del valor devuelto:

flask.Response

src.controllers.expense_controller.gastos()[fuente]

Muestra y gestiona los gastos de un usuario.

GET:

  • Lista los gastos del usuario seleccionado.

  • Permite cambiar de usuario mediante el parámetro user_id.

POST:

  • Crea un nuevo gasto tras validar los datos enviados desde el formulario.

Query user_id:

(opcional) ID del usuario a consultar. Default = 1.

Form producto:

Nombre del producto del gasto.

Form monto:

Monto asociado al gasto.

Form categoria:

Categoría del gasto.

Form es_hormiga:

Si está presente, el gasto se marca como «hormiga».

Devuelve:

Render de la plantilla gastos.html con: - user_id (int): ID del usuario activo. - gastos (list): Lista de gastos asociados. - categorias (list): Categorías válidas. - errores (dict): Diccionario de errores de validación (si los hay).

Tipo del valor devuelto:

flask.Response
