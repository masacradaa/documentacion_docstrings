src.repositories.file_security module

class src.repositories.file_security.FileSecurity[fuente]

Bases: object

Clase utilitaria para gestionar la seguridad de archivos dentro de la carpeta storage/.

Usa comandos de Windows (icacls y attrib) para aplicar permisos de lectura, escritura y protección frente a eliminación, asegurando que los archivos solo puedan ser manipulados por cuentas de administrador y el sistema.

Nada. Garantiza que la carpeta de almacenamiento exista.

static proteger_archivo(path: str) → None[fuente]

Aplica políticas de seguridad a un archivo:

  • Quita herencia de permisos a la clase padre Usuarios.

  • Concede permisos de lectura y escritura al grupo Usuarios.

  • Concede control total a Administradores y SYSTEM.

  • Quita permisos al grupo Todos.

  • Niega explícitamente la eliminación de archivos a Usuarios.

Parámetros:

path (str) – Ruta absoluta o relativa al archivo a proteger.

Devuelve:

Nada. Aplica permisos directamente en el sistema de archivos.

Repositorio para el manejo de persistencia de usuarios en el archivo usuarios.json.

Esta clase se encarga de: - Cargar los usuarios desde almacenamiento seguro. - Guardar usuarios aplicando las protecciones de seguridad mediante FileSecurity.
